# 📒 Phone Book

یک برنامه ساده دفتر تلفن که با زبان **Python** و کتابخانه **Tkinter** ساخته شده است.

## ✨ امکانات

- افزودن مخاطب جدید
- نمایش لیست مخاطبین
- جستجوی مخاطب
- حذف مخاطب
- رابط کاربری گرافیکی (GUI) با Tkinter

## 🛠️ تکنولوژی‌های استفاده شده

- Python 3
- Tkinter

## 🚀 نحوه اجرا

1. پروژه را دانلود یا Clone کنید:

```bash
git clone https://github.com/amirmehdiagah2211-coder/python_project.git
```

2. وارد پوشه پروژه شوید:

```bash
cd python_project
```

3. برنامه را اجرا کنید:

```bash
python main.py
```

> در صورتی که فایل اصلی پروژه نام دیگری دارد، به جای `main.py` همان فایل را اجرا کنید.

```
python_project/

```
